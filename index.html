What is an IP address?


An IP address is an identifier for a device that is connected to a network that uses the internet protocol.
It serves to route data packets across networks which allows devices to communicate with each other.
There are two types of IP address: IPV4 and IPV6 where IPv4 is a series of numbers separated by
periods while IPV6 is a series of numbers and letters separated by colons. This blog will only discuss
IPV4 thoroughly as it is the most common type of IP address used nowadays.


What is the IPV4 notation?


An IPV4 address is a 32 binary bit address written as four numbers separated by periods.
It is used to express the identifier of a device. The 32 bits are split into four groups
of numbers and each group represents an 8-bit segment called an octet. When converted
from bits to decimal, the number range in each octet is from 0 to 255.
An example of an IPV4 address is 192.168.10.128.


Back to IP addresses!


IP addresses consists of a network part and a host part so that networks can
be logically broken down into smaller networks called subnets
(smaller networks within a large network).


What is subnetting?


The process of breaking one big network down into smaller networks is called subnetting.
Subnetting is achieved by changing the default subnet mask by borrowing some bits from the host portion
of an IP address to create more available networks. The default subnet mask for each IP address class are listed below:


Class A Class B Class C
255.0.0.0 255.255.0.0 255.255.255.0


What is a subnet mask?


The subnet mask is 32 bit long. A subnet mask is assigned along with an IP address
represented in dotted decimal format. A subnet mask identifies which part of the IP address
relates to the network and which relates to the host. If we convert the decimals back into bits,
we can see which bits of the IP is allocated to the network and which are allocated to the host.

For example:

A subnet mask of 255.255.255.0 to bits will be
11111111.11111111.11111111.00000000
The 1's represents the network portion
while the 0’s represents the host portion.

What is host address?


The host address specifies the individual device within that network.
It represents as a unique identifier assigned to each device which can be
obtained by taking the remaining bits of the IP address after the network
portion has been determined.

For example:

A subnet mask of 255.255.255.0 to bits will be
11111111.11111111.11111111.00000000
The 1's represents the network portion
while the 0’s represents the host portion.


What is network address?


The network address specifies the network, it represents as an identifier
for a specific network / subnet. It is obtained by applying a subnet mask
to an IP address which determines the network segment to which the device belongs.

What is CIDR?



Remember about subnet mask? It can also be expressed in a different method
called a as CIDR(Classless Inter-Domain Routing) or slash notation.
255.255.255.0 subnet mask if expressed to CIDR will be a /24.
This means that the subnet mask is 24 bits in length meaning it has 24 of 1’s.
If the subnet mask is 255.255.255.192, then the CIDR is equivalent to /26.
